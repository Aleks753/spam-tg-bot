# Hedgehog in the Fog

![Hedgehog in the Fog](https://assets.mubi.com/images/film/22519/image-w1280.jpg)

### Запуск приложения

1. Установить зависимости

```
yarn
```

2. Создать `.env` файл по примеру

3. Убедиться, что бот имеет доступ к сообщениям в группах

4. Запустить бота

```
yarn start
```
